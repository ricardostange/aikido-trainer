# Aikido Trainer

## Project Overview
This project is a simple single-page HTML application designed to help users learn Aikido techniques.

## Usage
- Open the `index.html` file in any web browser to start using the application.
- Navigate through the interface to explore various Aikido techniques and tutorials.

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/ricardostange/aikido-trainer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd aikido-trainer
   ```
3. Open the `index.html` file in your browser.
